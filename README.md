# Dominio de Joins en SQL Server: Northwind & Pubs

## Descripción del Proyecto
Este proyecto consiste en el diseño y ejecución de 20 consultas SQL avanzadas utilizando juntas (Joins) en los entornos relacionales de Northwind y Pubs. Se enfoca en resolver problemas con lógica de negocio real.
